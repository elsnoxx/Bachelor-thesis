@startuml BalloonGame_SequenceDiagram
title Sekvenční diagram hry Balón

skinparam sequenceArrowThickness 1.5
skinparam responseMessageBelowArrow true
skinparam sequenceGroupBorderThickness 1
skinparam noteBorderThickness 1

actor "Hráči (2-4)" as Players
participant "React Frontend\n(BalloonGame)" as Frontend
participant "BLE Service\n(Senzor Sensatio)" as BLE
participant "SignalR Hub\n(GameHub)" as Hub
participant "Backend Service\n(BalloonGameService)" as Service
participant "Game Model\n(BalloonControlGame)" as Model
database "PostgreSQL\n(via DbWriteQueue)" as DB

== 1. Inicializace a připojení ==

Players -> Frontend : Otevře místnost (roomId)
Frontend -> Hub : Connect (Token)
Hub -> Hub : Authorize User (JWT)
Frontend -> Hub : JoinRoom(roomId)

Players -> Frontend : Párování senzoru Sensatio
Frontend -> BLE : Vyhledat zařízení (Web Bluetooth API)
BLE --> Frontend : Připojeno (isConnected = true)

== 2. Individuální kalibrace (10 s) ==


loop Každých 500 ms
    BLE -> Frontend : Surová hodnota EDA (v_raw)
    Frontend -> Frontend : v_norm = (v_raw / 65535) * 1000
    Frontend -> Hub : SendGameData(v_norm)
    Hub -> Service : ProcessCalibration(email, v_norm)
end
Service -> Model : Stanovit individuální baseline pro každého hráče

== 3. Závodní fáze (Real-time biofeedback) ==

loop Každých 500 ms dokud IsFinished == false
    BLE -> Frontend : Surová hodnota EDA (v_raw)
    Frontend -> Frontend : v_norm = (v_raw / 65535) * 1000
    Frontend -> Hub : SendGameData(roomId, "balloon", v_norm)
    Hub -> Service : ProcessInput(roomId, email, v_norm)

    group Výpočet pohybu na serveru
        Service -> Model : player.Altitude = v_norm
        
        alt Tonická složka: v_norm <= baseline
            Service -> Model : player.DistanceTraveled += 5
        end

        alt Fázický výkyv: Detekce SCR píku (> 750 jedn.)
            Service -> Model : player.DistanceTraveled += 20 (Bonus)
        end
        
        Service -> Model : Vypočti Progress (%)
    end

    Service -> DB : QueueBioFeedbackAsync(email, v_norm)
    Service --> Hub : ReceiveGameState(state)
    Hub --> Frontend : ReceiveGameState(state)
    
    Frontend -> Frontend : Aktualizuj UI:\n- X (progress)\n- Y (Altitude = v_norm)
end

== 4. Konec hry (Vítězství) ==

note over Service : player.DistanceTraveled >= 1000

Service -> Model : IsFinished = true, WinnerEmail = email
Service -> DB : QueueGameResultAsync(stats)
Service --> Hub : ReceiveGameState(finalState)
Frontend -> Players : Zobraz GameOverModal (Pořadí hráčů)

@enduml
